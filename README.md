Odin Project: Flexbox Assignment
Overview
This repository contains my solution for The Odin Project's Flexbox Assignment. The task is to recreate a static webpage from two provided design images: one showing the full layout, and another detailing fonts and colors. The focus is on practicing HTML structure, CSS styling, and flexbox layout techniques.
Features
HTML Structure: Semantic markup for all sections of the design.

CSS Styling: Custom styles to match the provided color scheme and typography.

Flexbox Layout: Responsive section alignment using flexbox (desktop view only).
